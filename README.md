# TiendaAnime

Proyecto académico desarrollado en Django que permite mostrar un catálogo de productos personalizados y gestionar pedidos enviados por los clientes desde un formulario web.

Este proyecto implementa un sistema web completo para una tienda de artículos personalizados inspirados en el mundo del anime.

Catálogo público de productos con imágenes.

Vista de detalle del producto.

Filtro por categorías.

Buscador de productos y categorías.

Formulario para solicitar pedidos, con subida de imagen.

Generación automática de un token único para seguimiento.

Página para ver el estado del pedido usando el token.

Productos destacados con imágenes.

Panel de administración de django:

Categorias.

Insumos.

Pedidos.

Productos.

Reseñas.

Funcionalidad extra:

Reseñas de productos por el cliente.

Instalación:

clonar repositorio

https://github.com/Roberto-Alfaro/TiendaAnime-git

Instalar dependencias:

pip install django - pillow

crear migraciones y aplicar:

python manage.py makemigrations y migrate

ejecutar servidor:

python manage.py runserver
